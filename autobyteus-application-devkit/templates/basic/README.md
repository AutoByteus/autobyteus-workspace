# __APPLICATION_NAME__

This is a custom AutoByteus application source project.

## Layout

- `application.json` declares the application manifest and generated runtime paths.
- `src/frontend/` contains editable browser source.
- `src/backend/` contains editable backend source, migrations, and assets.
- `dist/importable-package/` is generated by `autobyteus-app pack` and is the package root to import into AutoByteus.

## Commands

```bash
pnpm install
pnpm build
pnpm validate
pnpm dev
```

Do not import this source repository root into AutoByteus. Import `dist/importable-package` after packing.
